# 🌎 FIFA World Cup 2026 Dashboard

Um dashboard interativo desenvolvido no **Power BI** para análise da **Copa do Mundo FIFA 2026**. O projeto reúne informações sobre **grupos**, **seleções**, **partidas**, **datas**, **horários** e **cidades-sede**, proporcionando uma experiência intuitiva para consulta dos jogos da competição.

O objetivo é demonstrar técnicas de **modelagem de dados**, **Power Query**, **DAX**, **relacionamentos entre tabelas** e **design de dashboards** utilizando um tema esportivo.

### Recursos disponíveis

- 🔎 **Filtro por seleção**
  - Permite localizar rapidamente todas as partidas de uma equipe específica.

- 📅 **Filtro por data e horário**
  - Exibe somente os jogos correspondentes ao período selecionado.

- 📋 **Tabela de partidas**
  - Lista todos os confrontos disponíveis de acordo com os filtros aplicados.

- 🆚 **Visualização do confronto**
  - Exibe de forma destacada os dois times participantes da partida.

- 📍 **Cidade da partida**
  - Mostra a cidade onde o jogo será realizado.

- ⚡ **Atualização dinâmica**
  - Todos os elementos do painel são atualizados automaticamente conforme os filtros selecionados.
